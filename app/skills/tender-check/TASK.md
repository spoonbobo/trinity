# Check if tender document follows the gov tender handbook guideline

## Objective

Develop an AI tool to automate compliance checking of a **Resident Site Staff** tender document against the **Government Tender Handbook Guidelines**. 

**Scope & User Flow
1.** Input

- User uploads **1 tender document** Word document

2. Guideline Extraction

- AI analyzes the tender content.
- AI identifies and extracts relevant handbook guideline requirements applicable to the document

3. Compliance Checking

The AI must evaluate:

**A. Contradictions**

- Detect internal contradictions within the same document.
- (Out of scope: cross-document references.)

**B. Omissions**

- Identify required handbook clauses or requirements that are missing.

**C. Compliance Confidence Level**

- Provide confidence level per requirement.
- Show supporting proof (exact clause/page reference).

D. Anything else you think it’s important 

4. Output

The system must:

- Highlight compliant areas (with justification + evidence).
- Highlight non-compliant areas (with justification + evidence).
- Provide structured output (requirement → status → evidence → confidence).
